# JENZiQ - 賦能每一位健身專業人士與學員的夢想藍圖

JENZiQ (jenziq-app) 是一款專為現代健身、瑜珈及運動產業設計的全方位行動管理平台。我們的使命是透過科技簡化繁雜的行政管理，讓專業教練能專注於核心教學，並為學員創造極致的個人化成長路徑。

## 🌟 核心理念與願景

JENZiQ 不只是一個管理工具，它是連結運動熱情與數位效率的橋樑。我們追求：
- **效率極大化**：透過自動化的數據追蹤與課程管理，徹底告別傳統紙本作業。
- **成長視覺化**：讓學員的每一分努力都能被看見，每一份進步都有數據支持。
- **管理智慧化**：為管理者提供精準的據點動態分析，助力場館決策躍升。

## 🚀 平台核心功能

### 1. 教練儀表板 (Coach Dashboard)
- **課程排程管理**：即時更新授課行程，掌握所有待辦事項。
- **學生綁定系統**：一目了然看見負責的所有學員及其進度。
- **證照與專業展示**：整合個人證照庫，向世界展示你的專業實力。

### 2. 學員成長系統 (Student Dashboard)
- **等級與稱號**：創新的等級系統 (Student Level)，增加運動趣味性與回饋感。
- **營養與運動紀錄**：精確追蹤飲食與訓練數據，打造完美的個人日誌。
- **即時聯繫**：透過專屬通訊功能 (Inbox/Chatroom) 隨時與教練保持溝通。

### 3. 管理端分析中心 (Manager Center)
- **據點授課占比**：視覺化分析各分店課程比重（如 SUNNY 瑜珈、MU 沐光運動空間等）。
- **營運動態監控**：透明化的審查機制，確保所有教學品質與流程合規。

### 4. 專業工具箱 (Creative Tools)
- **AI 智能助手**：隨時隨地的 Support Bot 為用戶解答應用程式使用疑問。
- **電商整合 (Shopify)**：連結選品商場，滿足學員對運動週邊的需求。

## 🛠 技術架構

JENZiQ 採用最前沿的技術開發，確保極速反應與跨裝置穩定性：
- **Frontend**: React + Vite + Vanilla CSS (Aesthetics focused)
- **Backend**: Supabase (Real-time Database & Auth)
- **Infrastructure**: Git-driven development with premium component design patterns
